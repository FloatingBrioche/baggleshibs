# Baggleshibs 

Baggleshibs is command-line game similar to battleships for people who prefer the letters "g" and "b" to the letters "t" and "p".


## Getting Started

How to get started:

- Start your venv:
`python -m venv venv`

- Install the requirements:
`pip install -r requirements.txt`

- Check the tests and behaviour:
`pytest --testdox`

- Run the game:
`python run.py`
